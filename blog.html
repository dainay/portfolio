Description

The main goal of this project was to master the use of sessions for user authentication and role management on the website. To achieve this, it was necessary to create a blog. There was no obligation to create any UI, as it was entirely dedicated to backend development. However, I wanted to add some styling to it.

As an idea for the project, I assumed that Arthur Weasley, a character from the universe of books by J.K. Rowling, who is very interested in the lives of Muggles (non-magical people), could decide to create his own blog to explore something non-magical and share his interest with other wizards.

Thus, the administrator (Arthur) has additional interface settings thanks to session management, allowing him to add new articles directly on the site, delete them, edit content, and also remove comments. Additionally, he has access to the user management panel, where he can delete a user and their comments, delete comments from specific users, or delete a user while preserving their comments under the name "Deleted User."

Any other site user has access to reading articles and comments, which are hidden by default. Each user can register and gain the ability to add comments.
