Description

The aim of this project was to learn how to create GIF animations in Photoshop, which could serve as banners on websites. To achieve this, it was necessary to come up with any advertising campaign and then create a storyboard.

For this task, I chose an advertisement for a fictional online store that offers to find a gift for anyone on your behalf. The most challenging aspect of the project was the 5 MB limit for the final file. You can see the result on the right.
