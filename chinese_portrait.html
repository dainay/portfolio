Description

This project became the starting point for my exploration of development and my first page created with HTML, CSS, and JavaScript.

The idea behind the project was to present myself through associations with various objects and phenomenas. The project had no thematic or specific element requirements for the page, except for a form. This form should have been designed to add new blocks to the site with imformation from the site viditor. Consequently, I worked with JSON objects.

As this project was my initial endeavor, there were no stringent requirements for responsiveness or accessibility. The only condition was that the site should display information correctly between 768 and 1280 pixels.

It was also essential, as per the project guidelines, to work exclusively with Free or Creative Commons sources.

To introduce myself, I chose a broad range of phenomena and objects that could evoke various emotions. The primary visual presentation used the parallax effect. To add a touch of individuality to the design, I decided to incorporate videos for navigation buttons and created a Christmas mode that triggers music and a snow animation.
